### Low Computational Processing Pipeline for Delivery Robots


<a href="https://github.com/yourusername/your-repo" target="_blank" style="display:inline-block;">

- Mentored by <a href="https://iitb.irins.org/profile/51854">Prof. Kavi Arya</a> and mentors at ERTS Lab ( <a href="https://www.e-yantra.org/">e‑Yantra</a>  )
- Researched Outdoor Navigation for Delivery Robots in dynamic environments
- Trained deep‑learning models for road segmentation, object detection, and tracking on a custom <a href=" https://www.kaggle.com/eyantraiit/semantic-segmentation-datasets-of-indian-roads">IIT Bombay road dataset</a>. 
- Achieved 91% accuracy in segmentation tasks and improved real‑time object detection by 15%
- Developed an Efficient High‑Fidelity Map (EHFM) for bot localization using key environment features on an onboard computer
- Published research in the <a href="https://icaart.scitevents.org/?y=2022"><b>14th International Conference on Agents and Artificial Intelligence (ICAART) 2022</b></a>
